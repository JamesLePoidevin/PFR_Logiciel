FICHIER D'AIDE :

INSTALLATION DU LOGICIEL :
1) Se mettre dans le dossier APPLICATION, dans le terminal
2) Taper la commande : $make Qwante

LANCEMENT DU LOGICIEL :
1) Se mettre dans le dossier APPLICATION, dans le terminal
2) Taper la commande : $./Qwante 
3) *Lancer chmod a+x* , s'il y a une erreur d'autorisation

AIDE AUX MENUS :
Quand vous êtes sur un menu, entrez le chiffre correspondant à ce que vous souhaitez faire pour l'executer.
