FICHIER D'AIDE :

INSTALLATION DU LOGICIEL :
Avant de lancer l'installation il est nécessaire de télécharger l'application "feh". Pour cela, il faut rentrer la commande "sudo apt-get install feh" dans le terminal linux.
Pour l'installation sur windows avec l'application ubuntu, il faut télécharger "xming" et "feh", et lancer xming avant de démarrer le programme.
1) Se mettre dans le dossier APPLICATION, dans le terminal
2) Taper la commande : $make Qwante

LANCEMENT DU LOGICIEL :
1) Se mettre dans le dossier APPLICATION, dans le terminal
2) Taper la commande : $./Qwante 
3) *Lancer chmod a+x* , s'il y a une erreur d'autorisation

AIDE AUX MENUS :
Quand vous êtes sur un menu, entrez le chiffre correspondant à ce que vous souhaitez faire pour l'executer. 

! ATTENTION ! 
Le logiciel ne fonctionne qu'avec les textes xml codés en UTF8 !
